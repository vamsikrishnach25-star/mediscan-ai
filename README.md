# Mediscan AI (Frontend)

Mediscan AI is a frontend application for an AI-powered health report analysis platform. It focuses on providing a clean, responsive, and user-friendly interface for visualizing medical report insights.

## Tech Stack
- React.js (Vite)
- JavaScript
- Tailwind CSS
- HTML & CSS

## Features
- Responsive and modern UI
- Reusable React components
- Dashboard-style layout
- Clean and maintainable codebase

## Project Status
🚧 Frontend completed and available in this repository.  
Backend development is currently in progress.
